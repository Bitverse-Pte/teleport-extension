# TeleportWallet

 > A multi-chain & multi-ID Wallet

## develop

```bash
# 安装依赖
yarn

# 本地启动服务
yarn build:dev

# 打包生产
yarn build:pro
```

## 加载扩展程序

1. 本地打包，生成的 `dist` 目录
2. 打开 `chrome://extensions/` 页面，勾选右上角“开发者模式”
3. 选择左上角按钮“加载已解压扩展程序”，载入 `dist` 目录
